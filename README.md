# PinksVault

PinksVault is a web app where PinkPantheress fans can rank their favorite songs, save favorites, and write short reviews.

## Planned Features

- **User Authentication** — Create an account and log in to keep your data saved across sessions
- **Personal Song Rankings** — Rank PinkPantheress songs and build your own custom tier list
- **Favorite Songs List** — Save the tracks you love most to your personal favorites vault
- **Song Reviews** — Write short reviews on any song and share your thoughts with other fans
- **Community Rankings** — See how other fans are ranking songs and discover new favorites

## Tech Stack

- [React](https://react.dev/) — UI library
- [Vite](https://vite.dev/) — build tool and dev server

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
