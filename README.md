# PinksVault

PinksVault is an interactive fan platform dedicated to PinkPantheress, where users can explore her music, rank songs, save favorites, and share reviews with a community of fans.

## Live Site
pinksvault.netlify.app

---

## Overview

PinksVault is a full-stack web application that combines music discovery with personalized user interaction. Users can create accounts, log in, and persist their data across sessions using a real backend powered by Supabase.

The platform allows fans to engage more deeply with PinkPantheress’ music through rankings, favorites, and shared community insights.

---

## Features

### Authentication
- Secure user signup and login via Supabase Auth
- Persistent user sessions

### Song Rankings
- Rank PinkPantheress songs based on personal preference
- Rankings stored per user in the database

### Favorites System
- Save and manage favorite songs
- Data persisted to backend

### Reviews
- Write and store song reviews
- Access your past reviews anytime

### Community Leaderboard
- Aggregated rankings across users
- Discover trending songs within the fanbase

### Profile Page
- View your personal rankings, favorites, and activity

---

## Tech Stack

### Frontend
- **React 19** — UI framework
- **Vite** — Build tool and dev server
- **JavaScript (ES6+)**
- **React Context API** — Global state management

### Backend
- **Supabase** — Database, authentication, and backend services
- **PostgreSQL** — Relational database (via Supabase)
