# PinksVault

A fan web app for PinkPantheress lovers to rank songs, save favorites, and share short reviews.

## About

PinksVault is a community space built for PinkPantheress fans. Whether you want to obsess over her discography, share hot takes on your favorite tracks, or keep a personal archive of the songs that hit different — this is the place.

## Planned Features

- **User Accounts** — Sign up and log in to save your data across sessions
- **Song Rankings** — Build and share your personal tier list of PinkPantheress songs
- **Favorites Vault** — Bookmark tracks to your personal collection
- **Short Reviews** — Leave bite-sized takes (think: 280 characters or less) on any song
- **Community Feed** — Browse what other fans are saying and rating

## Tech Stack

- [React](https://react.dev/) + [Vite](https://vite.dev/) — frontend framework and build tool

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.
